## 🤔 What

A small React project showing how **TanStack Query (React Query)** handles server data like a boss 😎.
It demonstrates **data fetching, caching, dynamic query keys, and background refetching** with a simple event search feature.

---

## 🌍 Live Demo

👉 https://tanstack-query-project.netlify.app/

---

## ⚡ Features

* 🔄 Data fetching with **TanStack Query**
* 🧠 Smart caching (so your server doesn’t cry)
* 🔑 Dynamic query keys for search
* 🚀 Automatic background refetching
* 🧹 Cache cleanup using **staleTime** & **gcTime**
* 🔍 Event search functionality
